GitHub Hacker

A powerful CLI tool to manage multiple GitHub accounts for batch operations

pip install github-hacker

Features

Multi-Account Management

Login, logout, and manage multiple GitHub accounts with ease.

Batch Operations

Perform operations with all accounts at once - star, watch, fork repositories.

Easy Installation

Install via pip, pipx, or build from source. Works on all platforms.

Star/Unstar

Star or unstar repositories with one or all accounts.

Watch/Unwatch

Subscribe or unsubscribe from repository notifications.

Fork

Fork repositories to all your accounts instantly.

Installation

Install with pip (Recommended)

$ pip install github-hacker

Install with pipx

$ pipx install github-hacker

Install from source

$ git clone https://github.com/SSBun/GithubHacker.git
$ cd GithubHacker
$ pip install -e .

Quick Start

Login with GitHub Account

$ github-hacker login myaccount YOUR_GITHUB_TOKEN

List Accounts

$ github-hacker config list

Star a Repository (All Accounts)

$ github-hacker star owner/repo
# Or use full URL
$ github-hacker star https://github.com/owner/repo

Star with Specific Account

$ github-hacker star owner/repo -a myaccount

All Commands

Account Management
# Add account
github-hacker login <name> <token>

# Remove account
github-hacker logout <name>

# List accounts
github-hacker config list

# Export accounts
github-hacker config export accounts.json

# Import accounts
github-hacker config import accounts.json
Repository Operations
# Star a repository
github-hacker star <repo>

# Unstar a repository
github-hacker unstar <repo>

# Watch a repository
github-hacker watch <repo>

# Unwatch a repository
github-hacker unwatch <repo>

# Fork a repository
github-hacker fork <repo>

# Check status
github-hacker status <repo>

# Show repository info
github-hacker info <repo>
